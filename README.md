# umiriCopy

一个 Chrome 浏览器扩展，用于将 X (Twitter) 上的帖子复制到剪贴板（支持以图片形式导出）。

## 功能

- 在 X (Twitter) 页面上一键复制帖子内容
- 支持将帖子导出为图片
- Popup 弹窗界面
- Options 设置页面

## 技术栈

- Vue 3 + TypeScript
- Vite
- Chrome Extension Manifest V3
- html-to-image
